# 🚀 AI Website Redesign Pipeline

**Automated pipeline for finding local businesses with outdated websites and redesigning them using AI agents.**

## Quick Start

```bash
git clone https://github.com/Tigha66/ai-website-redesign-pipeline.git
cd ai-website-redesign-pipeline
python3 run_pipeline.py "nail salons" "Sydney, Australia"
```

## Pipeline Steps

1. step1_google_maps_scraper.py - Find businesses
2. step2_website_qualifier.py - AI assessment (1-10 score)
3. step3_website_redesigner.py - Generate modern HTML
4. step4_website_deployer.py - Deploy + email generation
