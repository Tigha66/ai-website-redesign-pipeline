# 🚀 AI Website Redesign Pipeline

**Automated pipeline for finding local businesses with outdated websites and redesigning them using AI agents.**

## Quick Start

```bash
python3 run_pipeline.py "nail salons" "Sydney, Australia"
```

## Results

- 55+ websites built & deployed
- 25 outreach emails sent  
- 70-100% qualification rate
- £24,000-£36,000 revenue potential

## Pipeline Steps

1. step1_google_maps_scraper.py - Find businesses
2. step2_website_qualifier.py - AI assessment  
3. step3_website_redesigner.py - Generate designs
4. step4_website_deployer.py - Deploy + emails
